## О себе
Я Python Backend-разработчик с сильным инженерным бэкграундом. К разработке пришёл через 10 лет работы с базами данных и ИТ-инфраструктурой в крупных организациях: видел, как софт автоматизирует процессы и избавляет людей от рутины. Это сформировало интерес к созданию систем, которые решают реальные задачи бизнеса.

Python выбрал за сочетание выразительности и промышленной мощности: на нём можно быстро собрать прототип и развернуть продакшен-сервис.
